\chapter{Hierarchical Clustering}


\section{Learning Objectives}

\begin{itemize}

\item Understand hierarchical clustering algorithm

\item Implement the algorithm
  
\item Learn the binary tree and its properties

\item Use the tree's properties to gain insight of data
  
\end{itemize}

\section{}

The {\it k-mean} clustering algorithm has third major problems: First,
there is no obvious reason to select a particular value of $k$.
Second, the algorithm is non-deterministic and it is often necessary
running the same program multiple times.  Third, it is possible to
calculate the distances of centroids but this requires additional
calculation; there is no easy way determining which two clusters are
closer than the others.
Figure~\ref{figure:kmean:distances} shows that the distances can vary
significantly in different executions even for the same $k$.  {\it
  Hierarchical clustering} is a method that does not require such a
pre-selected parameter.
